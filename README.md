# 🧠 Fundamental Deep Learning

Repository ini merupakan dokumentasi perjalanan saya dalam mempelajari ** Fundamental Deep Learning**.

Seluruh isi di dalam repository ini akan terus berkembang seiring dengan proses belajar yang saya lakukan, mulai dari memahami konsep dasar hingga implementasi dalam berbagai bentuk latihan dan proyek.

---

## 📌 Tentang Repository

Repository ini digunakan sebagai:

- Tempat menyimpan hasil pembelajaran
- Dokumentasi proses eksplorasi dan eksperimen
- Arsip latihan dan pengembangan proyek
- Bagian dari pengembangan portofolio di bidang AI / Machine Learning

Setiap bagian di dalam repository ini dapat memiliki pendekatan, struktur, dan tujuan yang berbeda sesuai dengan materi yang sedang dipelajari.
